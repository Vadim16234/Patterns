Шаблон: Абстрактная фабрика (Abstract Factory)

Цель:
Создание интерфейса, для создания множества взаимосвязных объектов, без жесткой привязки к конкретным классам

Для чего используется:
Для создания множеств взаимосвязных объектов.

Пример использования:
- система не должна зависеть от метода создания, компоновки и представления входящих в нее объектов;
- входящие взаимосвязные объекты должны использоваться вместе;
- система должна конфигурироваться одним из множеств объектов, из которых она состоит;
- нам необходимо представить множество объектов, раскрывая только их интерфейсы, но не реализацию.
